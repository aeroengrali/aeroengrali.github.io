GPS Single-Point Positioning + EKF

A from-scratch GNSS pipeline in C++17

Estimation & NavigationSoftware

A GPS-only pseudorange positioning engine in modern C++17: RINEX 3.x parsing, broadcast orbit/clock modelling, weighted least squares and an EKF, validated against precise IGS SP3 orbits with an independent Python audit.

Source on GitHub

Highlights

  • North RMS 0.54 m, horizontal ~0.93 m (public IGS station, GPS-only static)
  • Broadcast-vs-precise orbit 3D RMS 1.56 m; full-day EKF 3D RMS 3.14 m
  • RINEX 3.x observation/navigation parsing, ephemeris health/fit checks
  • Deterministic tests plus an independent Python results audit
